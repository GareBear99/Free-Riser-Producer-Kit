## 🎚️ Velocity Layers

Every one-shot comes in three velocity layers so your sampler responds dynamically:

| Suffix | Velocity | Use Case |
|--------|----------|----------|
| `_hard` | 100% | Main hits, accents, downbeats |
| `_mid`  | 65%  | Ghost notes, secondary hits |
| `_soft` | 35%  | Whisper hits, humanization layers |

Load all three into your sampler's velocity zones and your beats will breathe.

## 🎛️ DAW Setup

### FL Studio
1. Open **FPC** or drag one-shots into a **Drum Sequencer** channel
2. Right-click each pad → **Load Sample**
3. In FPC, use the velocity slider per step to use soft/mid/hard layers
4. Drag loops directly onto the **Playlist** — set BPM to match file name

### Ableton Live
1. Drag the `01_One_Shots` folder into a **Drum Rack**
2. Ableton auto-maps each file to a pad — rename to match your workflow
3. For velocity layers: drop `_soft`, `_mid`, `_hard` versions of the same hit onto the same pad → set velocity ranges
4. Drag loops into an **Audio Track**, right-click → **Warp** → set to session BPM

### Logic Pro X
1. Open **Drum Machine Designer** — drag one-shots onto pads
2. Or use **Ultrabeat** for velocity-layered programming
3. Drag loops into the timeline — use **Flex Time** if your project BPM differs

### MPC / Maschine
1. Load each category into a separate pad bank
2. Assign `_soft`, `_mid`, `_hard` versions to velocity zones within a single pad
3. Use the loop files as reference patterns while programming your own sequences

## 💡 Production Tips

- **Kick + Sub layering** — Stack `Kick_Trap` (attack) with `Kick_SubOnly` (body) at matched levels. This is how professional trap kicks are built. The snap and the sub occupy different frequency ranges and never fight.
- **Ghost note snares** — Use `Snare_Fat_soft` or `Snare_Vinyl_soft` between your main snare hits at -8 to -12dB. This is what separates static patterns from grooves that feel live.
- **Clap doubling** — Layer `Clap_Tight` with `Clap_Reverb` for a clap that hits hard upfront and breathes in the room. Pan Tight slightly L, Reverb slightly R.
- **Hat humanisation** — Alternate between `Hat_Closed_hard` and `Hat_Closed_mid` on consecutive 16ths. Humans never hit hats at the same velocity twice.
- **Conga + Shaker groove** — Program `Conga_Hi` on the "e" and "ah" of beats 2 and 4 with `Shaker` running constant 16ths underneath. Instant Afrobeats sub-groove.
- **808 Tom fills** — `Tom_808` has an 808-style pitch drop, which means it works as a mini 808 hit for fills and transitions without needing to retune a bass sample.
- **Crash on bar 1 only** — `Crash_Bright_soft` at -8dB on bar 1 of every 4-bar section adds a sense of structure without being obvious. Mix engineers call this the "invisible crash."
- **Sidechain everything to the kick** — Your 808, synths, and pads should all duck slightly when `Kick_Deep` hits. 3–5dB of gain reduction with a fast attack (0.1ms) and medium release (80ms) is all you need.

## 🎚️ Mixing Reference

| Element | EQ Tip | Compression | Level |
|---------|--------|-------------|-------|
| Kick | Boost 60Hz for sub, cut 300–500Hz mud, boost 3–5kHz click | 4:1, fast attack, medium release | 0 dBFS (reference) |
| Snare | Boost 200Hz for body, boost 5–8kHz for crack | 4:1, medium attack to preserve snap | −3 to −6 dBFS |
| Clap | High-pass at 200Hz, boost 2–4kHz presence | Light limiting only | −4 to −6 dBFS |
| Hi-Hats | High-pass at 6kHz, gentle shelf boost above 10kHz | None or very light | −8 to −12 dBFS |
| Cymbals | High-pass at 2kHz, cut harsh 5–7kHz if needed | Light limiting | −10 to −14 dBFS |
| Percs | Shape to taste, cut below 200Hz unless it's a tom | Light glue compression | −10 to −16 dBFS |
